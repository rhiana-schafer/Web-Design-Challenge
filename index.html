In this project, weather patterns from over 500 cities across the world were analyzed using Pandas and Python.

These data were collected from OpenWeatherMap API, and include:

  • Temperature
  • Humidity
  • Cloud Cover
  • Wind Speed


Displayed here are plots generated by MatPlotLib, where you can see how city latitude affects weather.
