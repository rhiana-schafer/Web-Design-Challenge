Weather and Latitude

In this project, I analysed weather (temperature, humidity, cloud cover, and wind speed) in over 500 cities across the world, using data from OpenWeatherMap API.
I used Matplotlib to visualize various patterns. Displayed here, you can see how city latitude affects weather. You can also reference the source data used to generate these plots.
